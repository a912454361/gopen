- 修复若干已知问题

## 📥 下载说明

### Windows
1. 下载 `gopen-windows-x64-v1.0.3.zip`
2. 解压后运行 `安装程序.exe`
3. 按提示完成安装

### Android
1. 下载 `gopen-android-v1.0.3.tar.gz`
2. 参考 `Android构建指南.md` 本地构建APK
3. 或使用 EAS 云端构建

### iOS
1. 下载 `gopen-ios-v1.0.3.tar.gz`
2. 需要 Mac + Xcode 环境
3. 参考 `iOS构建指南.md` 构建

### macOS
1. 下载 `gopen-macos-v1.0.3.tar.gz`
2. 参考 `macOS构建指南.md` 构建

### Linux
1. 下载 `gopen-linux-v1.0.3.tar.gz`
2. 解压后可直接运行 AppImage
3. 或安装 DEB/RPM 包

### Web
1. 下载 `gopen-web-v1.0.3.tar.gz`
2. 部署到任意静态服务器
3. 支持 Vercel/Netlify/GitHub Pages
