## 三、截图制作方法

### 方法1：使用截图生成器（推荐）

**步骤**：
1. 打开文件：`release-package/02-应用截图/screenshot-generator.html`
   ```bash
   # macOS
   open release-package/02-应用截图/screenshot-generator.html
   
   # Windows
   start release-package/02-应用截图/screenshot-generator.html
   
   # Linux
   xdg-open release-package/02-应用截图/screenshot-generator.html
   ```

2. 浏览器会自动打开HTML文件

3. 点击每个截图下方的"📥 下载截图"按钮

4. 系统会自动生成并下载PNG格式截图

5. 保存到指定位置

**优点**：
- ✅ 自动生成标准尺寸
- ✅ 高质量输出
- ✅ 无需设计技能

---

### 方法2：使用真实设备截图

**Android设备截图**：
1. 在Android设备上安装应用
2. 打开应用，切换到对应页面
3. 同时按住"电源键" + "音量减键"
4. 截图会保存到相册

**iOS设备截图**：
1. 在iPhone/iPad上安装应用
2. 打开应用，切换到对应页面
3. 同时按住"电源键" + "音量加键"
4. 截图会保存到相册

**优点**：
- ✅ 展示真实应用界面
- ✅ 用户信任度更高

---

### 方法3：使用模拟器截图

**Android模拟器**：
```bash

# 然后运行应用
```

**iOS模拟器**：

## 五、截图质量要求

### ✅ 必须满足
- [ ] 图片清晰，无模糊
- [ ] 分辨率符合要求
- [ ] 无明显色差
- [ ] 无水印或Logo遮挡
- [ ] 文字清晰可读

### ⚠️ 注意事项
- 不要使用真实用户数据
- 不要展示敏感信息
- 不要包含第三方品牌Logo
- 界面要有内容，不要空白
