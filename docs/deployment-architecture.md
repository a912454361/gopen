                    └──────────────┬──────────────────────┘
                                   │
              ┌────────────────────┴────────────────────┐
              │                                         │
              ▼                                         ▼
┌─────────────────────────┐             ┌─────────────────────────┐
│    阿里云 ECS (当前)     │             │   腾讯云 SCF (生产)     │

## 阶段规划

### 阶段一：当前（立即可用）
- 使用阿里云 ECS IP 直连
- 地址：`http://8.136.229.243:9091`
- 用途：开发测试、临时访问

### 阶段二：短期（1-3天）
- 阿里云 ECS + SSL 证书
- 地址：`https://gopen.com.cn`（通过阿里云）
- 用途：正式上线

### 阶段三：长期（备案后）
- 腾讯云 SCF + 函数 URL
- 地址：`https://gopen.com.cn`（通过腾讯云）
- 用途：生产环境、弹性扩展

---

## 平台对比

| 特性 | 阿里云 ECS | 腾讯云 SCF |
|------|-----------|-----------|
| 备案状态 | ✅ 已备案 | ⏳ 需备案 |
| 上线时间 | ✅ 立即可用 | ⏳ 3-20天 |
| 费用 | 按月固定 | 按量计费 |
| 弹性 | 需手动扩容 | 自动弹性 |
| 维护 | 需运维 | 免运维 |
| 适合场景 | 稳定负载 | 流量波动 |

## 域名解析规划

```
gopen.com.cn        → 阿里云 ECS（备案后切到腾讯云）
api.gopen.com.cn    → 阿里云 ECS（API 子域名）
test.gopen.com.cn   → 阿里云 ECS（测试环境）
prod.gopen.com.cn   → 腾讯云 SCF（生产环境）
```
