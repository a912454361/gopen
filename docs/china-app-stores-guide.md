# 国内应用市场提交指南

## 一、华为应用市场

### 1.1 注册开发者账号
1. 访问 https://developer.huawei.com/consumer/cn/
2. 点击"注册"按钮
3. 使用手机号注册
4. 完成实名认证（个人/企业）

### 1.2 创建应用
1. 进入"管理中心"
2. 点击"创建应用"
3. 填写应用基本信息：
   - 应用名称：G open 智能创作助手
   - 应用包名：com.gopen.ai.assistant
   - 应用类型：应用
   - 一级分类：工具
   - 二级分类：效率

### 1.3 上传应用
1. 上传 APK 文件
2. 填写版本信息
3. 上传应用图标（512x512）
4. 上传应用截图（至少 2 张）

### 1.4 填写应用信息
- 应用简介：G open 是一款集成多家顶级 AI 模型的智能创作助手...
- 应用描述：详细描述应用功能
- 更新日志：V1.0 首次发布
- 隐私政策：https://woshiguotao.cn/privacy-policy

### 1.5 提交审核
1. 确认信息无误
2. 点击"提交审核"
3. 等待审核结果（1-3 天）

## 六、常见问题

### Q1: 审核被拒绝怎么办？
A: 查看拒绝原因，修改后重新提交。常见原因：
- 应用描述包含敏感词
- 截图不符合要求
- 缺少隐私政策
- 权限说明不清晰

### Q2: 需要软著证书吗？
A: 部分平台需要：
- 腾讯应用宝：需要
- 360 手机助手：需要
- 其他平台：建议准备

### Q3: 审核周期多久？
A: 各平台不同：
- 华为、小米、OPPO、vivo：1-3 天
- 腾讯应用宝：3-7 天
- 360、百度：3-7 天
