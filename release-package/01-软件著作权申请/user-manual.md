## 一、软件功能说明

### 1.1 软件概述

G open 是一款基于人工智能技术的智能创作助手应用，集成多个顶级AI模型，为用户提供对话创作、图像生成、项目管理等功能。软件采用前后端分离架构，前端使用 Expo React Native 开发，后端使用 Node.js Express 开发，支持 Android、iOS 和 Web 三端运行。

### 1.2 核心功能模块

#### 1.2.1 AI对话创作模块

**功能描述**：
集成 GPT-4o、Claude 3、豆包等多个AI大模型，支持智能对话和内容创作。

**主要功能**：
- 多模型切换：支持在多个AI模型间自由切换
- 上下文记忆：最长支持 200K Token 上下文
- 流式输出：实时显示AI生成内容
- 对话管理：保存和管理历史对话记录
- 创作模板：内置多种创作场景模板

**技术实现**：
- 使用 Server-Sent Events (SSE) 实现流式传输
- 采用 Token 计费系统，精确统计使用量
- 支持自定义 API 密钥配置

#### 1.2.2 图像生成模块

**功能描述**：
集成 DALL-E 3、Midjourney 等图像生成模型，支持文本生成图像。

**主要功能**：
- 文本描述生成图像
- 图像风格定制
- 图像尺寸选择
- 批量生成
- 图像编辑和优化

**技术实现**：
- 调用 OpenAI DALL-E API
- 支持 Base64 和 URL 两种图像格式
- 自动保存生成记录

#### 1.2.3 项目管理模块

**功能描述**：
提供项目创建、管理、统计等功能，帮助用户高效管理创作项目。

**主要功能**：
- 项目创建和分类
- 项目进度跟踪
- 资源消耗统计
- Token 使用分析
- 项目数据导出

**技术实现**：
- 使用 Supabase 数据库存储项目数据
- 实时统计 Token 消耗
- 支持多种图表可视化

#### 1.2.4 云存储同步模块

**功能描述**：
支持多个云存储平台，实现项目文件的自动同步和备份。

**主要功能**：
- 支持百度网盘、阿里云盘、Google Drive 等
- 自动同步项目文件
- 文件版本管理
- 离线下载

**技术实现**：
- OAuth 2.0 授权认证
- RESTful API 调用
- 增量同步机制

#### 1.2.5 会员管理模块

**功能描述**：
提供三级会员体系，包括免费用户、普通会员和超级会员。

**主要功能**：
- 会员权益展示
- 在线支付开通
- 会员到期提醒
- 权限自动控制

**技术实现**：
- 微信支付/支付宝支付集成
- 自动审核充值订单
- 定时任务检查会员状态

#### 1.2.6 API密钥管理模块

**功能描述**：
允许用户配置自己的 API 密钥，使用专属模型服务。

**主要功能**：
- 多平台密钥管理
- 密钥加密存储
- 密钥有效性验证
- 使用记录追踪

**技术实现**：
- AES-256 加密存储
- 密钥脱敏显示
- 定时验证密钥有效性

## 二、系统架构说明

### 2.1 技术架构

```
┌─────────────────────────────────────────┐
│           客户端层 (Client)              │
│  ┌──────────┬──────────┬──────────┐    │
│  │ Android  │   iOS    │   Web    │    │
│  └──────────┴──────────┴──────────┘    │
│         Expo React Native               │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│           API 网关层 (Gateway)          │
│         Express.js + Middleware         │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│           业务服务层 (Service)           │
│  ┌──────────┬──────────┬──────────┐    │
│  │ AI服务   │ 用户服务 │ 支付服务 │    │
│  ├──────────┼──────────┼──────────┤    │
│  │ 项目服务 │ 存储服务 │ 会员服务 │    │
│  └──────────┴──────────┴──────────┘    │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│           数据存储层 (Storage)           │
│  ┌──────────┬──────────┬──────────┐    │
│  │ Supabase │ 对象存储 │ 缓存服务 │    │
│  └──────────┴──────────┴──────────┘    │
└─────────────────────────────────────────┘
```

### 2.2 技术栈

#### 前端技术
- **框架**：Expo 54 + React Native
- **路由**：Expo Router
- **状态管理**：React Context + Hooks
- **UI组件**：自定义组件库
- **样式**：StyleSheet + 主题系统

#### 后端技术
- **运行时**：Node.js 24
- **框架**：Express.js
- **数据库**：PostgreSQL (Supabase)
- **对象存储**：S3 兼容存储
- **AI SDK**：coze-coding-dev-sdk

#### 第三方服务
- **AI 模型**：OpenAI、Anthropic、豆包
- **支付**：微信支付、支付宝
- **云存储**：百度网盘、阿里云盘、Google Drive

## 四、安装部署说明

### 4.1 客户端安装

**Android 安装**：
1. 从 Google Play 或应用市场下载 APK
2. 允许安装未知来源应用
3. 点击安装完成

**iOS 安装**：
1. 从 App Store 搜索"G open"
2. 点击"获取"按钮
3. 验证 Face ID/Touch ID 完成

**Web 访问**：
1. 打开浏览器访问 https://woshiguotao.cn
2. 注册或登录账户
3. 开始使用

### 4.2 服务器部署

```bash
# 1. 克隆代码仓库
git clone https://github.com/gopen/gopen-app.git

# 2. 安装依赖
cd server && pnpm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填写必要配置

# 4. 初始化数据库
pnpm run db:init

# 5. 启动服务
pnpm run start
```

## 五、操作说明

### 5.1 用户注册与登录

1. 打开应用，点击"注册"按钮
2. 输入邮箱或手机号
3. 设置密码（至少8位，包含字母和数字）
4. 完成验证码验证
5. 注册成功，自动登录

### 5.2 AI对话操作

1. 在首页选择"AI对话"功能
2. 从模型列表选择AI模型
3. 在输入框输入问题或指令
4. 点击发送，实时查看AI回复
5. 可保存对话或继续对话

### 5.3 会员开通流程

1. 进入"会员中心"页面
2. 选择会员类型（普通/超级）
3. 点击"立即开通"
4. 选择支付方式（微信/支付宝）
5. 扫码完成支付
6. 系统自动激活会员

### 5.4 项目管理操作

1. 点击"项目管理"进入列表
2. 点击"新建项目"创建项目
3. 填写项目名称和描述
4. 选择项目类型
5. 开始创作和管理

## 六、技术特点

### 6.1 创新性

1. **多模型融合**：集成多个顶级AI模型，用户可自由切换
2. **流式输出**：采用SSE技术实现实时流式输出
3. **多端同步**：支持Android、iOS、Web三端数据同步
4. **自动计费**：精确统计Token使用量，自动扣费

1. **数据加密**：所有敏感数据采用 AES-256 加密

### 6.3 可扩展性

1. **模块化设计**：各功能模块独立，易于扩展
2. **插件机制**：支持第三方插件集成
3. **API开放**：提供完整的API接口
4. **微服务架构**：支持服务拆分和扩展
