## 材料格式要求

### 源代码文档要求

1. **页数要求**：共60页（前30页 + 后30页）
2. **每页行数**：不少于50行
3. **页眉格式**：右上角标注页码和软件名称版本
   - 格式：`第X页                          G open V1.0`
4. **字体要求**：
   - 标题：宋体或黑体，小四号
   - 正文：宋体，小四号
5. **纸张规格**：A4纸，单面打印
6. **注意事项**：
   - 代码中不能包含敏感信息（API密钥、密码、真实密钥等）
   - 已自动过滤敏感信息

### 软件说明书要求

1. **页数要求**：不少于10页
2. **内容要求**：
   - 软件功能说明
   - 操作流程说明
   - 界面截图
   - 技术特点
3. **字体要求**：
   - 标题：黑体，三号
   - 正文：宋体，小四号
4. **纸张规格**：A4纸，单面打印

### 身份证复印件要求

1. 正反面复印在同一张A4纸上
2. 图像清晰，文字可读
3. 本人签字确认

## 提交方式

### 方式一：中国版权保护中心（官方）

1. 访问：https://www.ccopyright.com.cn/
2. 注册账号并登录
3. 在线填写申请表
4. 上传材料
5. 邮寄纸质材料
6. 等待审核

### 方式二：代理机构（推荐加急）

#### 阿里云软著服务
1. 访问：https://tm.aliyun.com/
2. 选择"软件著作权"
3. 选择加急套餐
4. 在线填写信息
5. 上传材料
6. 支付费用
7. 等待审核

#### 腾讯云知识产权
1. 访问：https://ipr.cloud.tencent.com/
2. 选择"软件著作权"
3. 填写申请信息
4. 上传材料
5. 支付费用
