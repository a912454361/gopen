## 第一步：注册账号

### 1.1 访问官网

打开浏览器，访问：**https://www.ccopyright.com.cn/**

### 1.2 点击注册

1. 点击右上角「注册」按钮
2. 选择「个人用户」
3. 填写注册信息：
   - 用户名（自定义）
   - 密码
   - 真实姓名
   - 身份证号
   - 手机号
   - 验证码
4. 点击「提交注册」
5. 完成手机验证

### 1.3 实名认证

1. 登录账号
2. 进入「用户中心」
3. 点击「实名认证」
4. 上传身份证正反面照片
5. 等待审核（一般1-2个工作日）

1. 检查所有信息是否正确

## 第三步：准备纸质材料

### 3.1 打印申请表

提交申请后，系统会生成申请表：

1. 在「我的申请」中找到刚提交的申请
2. 点击「打印申请表」
3. 用A4纸打印
4. 在「申请人签章」处**亲笔签字**
5. 签字日期填写当天日期

### 3.2 打印源代码文档

```
文件位置：docs/软件著作权申请材料/
├── 源代码文档-前30页.txt  → 打印
└── 源代码文档-后30页.txt  → 打印

打印要求：
- A4纸单面打印
- 共60页
- 不要装订
- 用回形针固定
```

### 3.3 打印软件说明书

```
文件位置：docs/软件著作权申请材料/软件说明书.md

打印要求：
- A4纸单面打印
- 建议用Word打开后调整格式再打印
- 标题用黑体三号
- 正文用宋体小四号
```

### 3.4 身份证复印件

1. 身份证正反面复印在同一张A4纸上
2. 确保清晰可读
3. 在复印件上签字

## 第五步：等待审核

### 5.1 审核流程

```
提交申请 → 形式审查 → 实质审查 → 登记公告 → 发放证书
   ↓           ↓           ↓           ↓           ↓
  即时      5-10天      40-60天      5天        5天
```

### 5.2 查询进度

1. 登录中国版权保护中心网站
2. 进入「用户中心」
3. 点击「我的申请」
4. 查看申请状态

### 5.3 状态说明

| 状态 | 说明 |
|------|------|
| 受理中 | 材料已收到，正在形式审查 |
| 补正 | 材料有问题，需补充修改 |
| 审查中 | 材料合格，正在实质审查 |
| 登记公告 | 审查通过，等待公告 |
| 已登记 | 完成，证书寄出 |

### 5.4 如需补正

如果收到「补正」通知：

1. 登录系统查看补正原因
2. 按要求修改材料
3. 重新提交或邮寄

## 常见问题

### Q1: 真的完全免费吗？
**A:** 是的，通过中国版权保护中心官网申请，普通件完全免费。不需要支付任何费用。

### Q2: 可以加急吗？
**A:** 官方也提供加急服务（300元），但您选择免费就要等60-90工作日。

### Q3: 申请期间可以修改软件吗？
**A:** 可以，不影响已提交的申请。

### Q4: 个人申请和企业申请有什么区别？
**A:** 
- 个人申请：用身份证，软件著作权归个人所有
- 企业申请：用营业执照，著作权归公司所有
- 两者法律效力相同

### Q5: 材料必须邮寄吗？
**A:** 是的，中国版权保护中心需要纸质材料存档，必须邮寄。

### Q6: 可以代办吗？
**A:** 可以委托他人代办，需要提供授权委托书。
