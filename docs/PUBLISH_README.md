## 📦 第一步：代码上传

### 1. 安装微信开发者工具
下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目
```bash
# 项目目录
miniprogram/

# 配置文件
miniprogram/project.config.json
```

### 3. 修改AppID
```json
// miniprogram/project.config.json
{
  "appid": "你的小程序AppID",
  ...
}
```

### 4. 上传代码
- 点击右上角「上传」
- 版本号：`1.0.0`
- 备注：`首发版本 - 横屏3D粒子仙侠游戏`

---

## 🌐 第二步：配置服务器域名

### 登录微信公众平台
地址：https://mp.weixin.qq.com/

### 配置路径
```
开发 → 开发管理 → 开发设置 → 服务器域名
```

### 需要配置的域名

| 类型 | 域名 |
|------|------|
| request合法域名 | `https://your-domain.com` |
| uploadFile合法域名 | `https://your-domain.com` |
| downloadFile合法域名 | `https://your-domain.com` |
| socket合法域名 | `wss://your-domain.com` |

### 域名要求
- ✅ 必须是HTTPS协议
- ✅ 必须有有效SSL证书
- ✅ 域名需ICP备案
- ❌ 不支持IP地址
- ❌ 不支持端口号

---

## 🔍 第三步：提交审核

### 1. 进入版本管理
```
管理 → 版本管理 → 开发版本
```

### 2. 点击「提交审核」

### 3. 填写审核信息

#### 基本信息
```
小程序名称：剑破苍穹
服务类目：游戏 → 休闲游戏
```

#### 功能页面（需填写5个）

| 页面 | 路径 | 功能描述 |
|------|------|----------|
| 主页面 | pages/main/main | 游戏主界面，3D粒子特效展示 |
| 战斗 | pages/battle/battle | 横屏战斗场景，技能对决 |
| 剧情 | pages/index/index | 互动剧情选择，分支故事 |
| 竞技场 | pages/arena/arena | PVP对战，积分排名 |
| 关卡 | pages/stage/stage | 关卡选择，扫荡碾压 |

#### 测试账号
```
测试方式：无需登录
说明：游戏自动分配UID，可直接体验
```

## ⚠️ 常见问题

### Q1：审核被拒绝怎么办？
查看拒绝原因，修改后重新提交。

常见拒绝原因：
- 功能不完整
- 内容违规
- 侵犯版权
- 缺少用户协议

### Q2：iOS端支付限制？
iOS端不能有虚拟支付入口：
- 隐藏iOS端充值按钮
- 或改为「联系客服」

### Q3：审核需要多久？
通常1-7个工作日

### Q4：如何加急审核？
紧急情况可申请「审核加急」
