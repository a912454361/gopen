# 万古长夜小程序上架指南

## 项目概述

本仓库包含万古长夜游戏平台在三个主流小程序平台的版本：

- `miniprogram/wechat/` - 微信小程序
- `miniprogram/alipay/` - 支付宝小程序
- `miniprogram/douyin/` - 抖音小程序
- `miniprogram/shared/` - 公共工具库

## 一、微信小程序上架

### 1. 准备工作

1. **注册小程序账号**
   - 访问 [微信公众平台](https://mp.weixin.qq.com/)
   - 注册小程序账号（选择"游戏"类目需游戏版号，建议选择"工具-娱乐"）
   - 完成主体认证

2. **获取 AppID**
   - 登录小程序后台 → 开发 → 开发设置 → AppID
   - 修改 `miniprogram/wechat/project.config.json` 中的 `appid`

3. **配置服务器域名**
   - 小程序后台 → 开发 → 开发设置 → 服务器域名
   - 添加 `request` 合法域名：`https://api.gopen.com`
   - 添加 `webview` 业务域名：`https://game.gopen.com`

### 2. 开发与调试

```bash
# 安装微信开发者工具
# 导入项目：miniprogram/wechat

# 配置业务域名（WebView 使用）
# 1. 登录小程序后台
# 2. 开发 → 开发设置 → 业务域名
# 3. 下载校验文件放到 game.gopen.com 根目录
# 4. 添加域名：game.gopen.com
```

### 3. 提交审核

1. 微信开发者工具 → 上传代码
2. 小程序后台 → 版本管理 → 开发版本 → 提交审核
3. 填写审核信息：
   - 服务类目：娱乐 → 其他娱乐
   - 测试账号：提供测试账号密码
   - 功能页面：首页、对战页、卡牌页

### 4. 注意事项

- ⚠️ WebView 加载的 H5 页面必须使用 HTTPS
- ⚠️ 业务域名需在后台配置并校验
- ⚠️ 虚拟支付需使用微信支付（iOS 禁止虚拟物品购买）
- ⚠️ 游戏类目需版号，建议使用"娱乐"类目

## 二、支付宝小程序上架

### 1. 准备工作

1. **注册小程序账号**
   - 访问 [支付宝开放平台](https://open.alipay.com/)
   - 创建小程序
   - 完成企业认证

2. **获取 AppID**
   - 小程序后台 → 设置 → 基本设置 → APPID
   - 修改 `miniprogram/alipay/mini.project.json` 中的 `appid`

3. **配置服务器域名**
   - 小程序后台 → 设置 → 域名设置
   - 添加接口域名：`api.gopen.com`
   - 添加 WebView 域名：`game.gopen.com`

### 2. 开发与调试

```bash
# 安装支付宝开发者工具
# 导入项目：miniprogram/alipay

# 配置能力
# 1. 在 app.json 中声明需要的权限
# 2. 配置支付宝支付能力
```

### 3. 提交审核

1. 支付宝开发者工具 → 上传
2. 小程序后台 → 版本管理 → 提交审核
3. 填写审核信息

### 4. 注意事项

- ⚠️ 支付宝小程序对 WebView 限制较多
- ⚠️ 建议主要使用原生页面，WebView 作为补充
- ⚠️ 支付能力需要开通支付宝商家服务

## 三、抖音小程序上架

### 1. 准备工作

1. **注册小程序账号**
   - 访问 [抖音开放平台](https://developer.open-douyin.com/)
   - 创建小程序
   - 完成企业认证

2. **获取 AppID**
   - 小程序后台 → 开发 → 开发设置 → AppID
   - 修改 `miniprogram/douyin/project.config.json` 中的 `appid`

3. **配置服务器域名**
   - 小程序后台 → 开发 → 开发设置 → 服务器域名
   - 添加域名：`api.gopen.com`, `game.gopen.com`

### 2. 开发与调试

```bash
# 安装抖音开发者工具
# 导入项目：miniprogram/douyin

# 特色功能
# 1. 视频广告激励（看广告领金币）
# 2. 抖音登录
# 3. 抖音支付
```

### 3. 提交审核

1. 抖音开发者工具 → 上传
2. 小程序后台 → 版本管理 → 提交审核

### 4. 注意事项

- ⚠️ 抖音小程序审核周期较长（3-7天）
- ⚠️ 游戏类目需要版号
- ⚠️ WebView 功能受限，建议原生开发

## 四、公共配置

### 环境变量

| 变量 | 说明 | 示例值 |
|------|------|--------|
| API_BASE_URL | 后端 API 地址 | https://api.gopen.com/api/v1/ink |
| GAME_WEB_URL | H5 游戏地址 | https://game.gopen.com/game |

### 后端 API 需要支持

```
POST /api/v1/ink/auth/wechat    - 微信登录
POST /api/v1/ink/auth/alipay    - 支付宝登录
POST /api/v1/ink/auth/douyin    - 抖音登录
POST /api/v1/ink/pay/wechat     - 微信支付
POST /api/v1/ink/pay/alipay     - 支付宝支付
POST /api/v1/ink/pay/douyin     - 抖音支付
```

## 五、审核常见问题

### Q1: 游戏类目需要版号怎么办？
A: 建议使用"娱乐-其他娱乐"或"工具-游戏工具"类目，避开版号要求。

### Q2: WebView 页面白屏？
A: 检查业务域名配置是否正确，H5 页面是否使用 HTTPS。

### Q3: iOS 端无法支付？
A: 微信小程序 iOS 端禁止虚拟物品购买，可引导用户到 H5 版本支付。

### Q4: 审核被拒？
A: 常见原因：
- 功能不完整
- 存在 Bug
- 涉及虚拟货币交易
- 涉及游戏版号
