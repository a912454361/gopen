## 四、关键准备事项

### 🔑 必需证书

1. **软件著作权证书**
   - 申请周期: 2-3 个月
   - 费用: 300-1000 元
   - 申请途径: 中国版权保护中心官网
   - 加急服务: 3-5 个工作日（费用更高）

2. **ICP 备案**
   - 申请周期: 20-30 天
   - 费用: 免费
   - 申请途径: 工信部备案系统
   - 当前状态: 已提交，等待审核

3. **隐私政策**
   - 必须包含: 数据收集说明、使用目的、用户权利
   - 托管地址: 应用内 + 公网 URL
   - 模板参考: https://www.freeprivacypolicy.com
