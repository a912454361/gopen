# 🚀 G open 企业官网部署指南

## 域名: woshiguotao.cn

---

## 方案一：Vercel 部署（推荐）

### 步骤 1: 下载网站文件
网站文件位于: `/workspace/projects/gopen-website.tar.gz`

### 步骤 2: 在本地解压
```bash
tar -xzvf gopen-website.tar.gz
cd website
```

### 步骤 3: 安装 Vercel CLI
```bash
npm i -g vercel
```

### 步骤 4: 登录 Vercel
```bash
vercel login
```
按提示选择 GitHub 登录

### 步骤 5: 部署
```bash
vercel --prod
```

### 步骤 6: 添加自定义域名
1. 访问 https://vercel.com/dashboard
2. 选择刚部署的项目
3. Settings → Domains
4. 添加 `woshiguotao.cn`
5. 添加 `www.woshiguotao.cn`

### 步骤 7: 配置域名解析
在域名服务商（阿里云/腾讯云）添加:

| 记录类型 | 主机记录 | 记录值 |
|---------|---------|--------|
| A | @ | 76.76.21.21 |
| CNAME | www | cname.vercel-dns.com |

## 方案二：Netlify 部署

### 步骤 1-2: 同上

### 步骤 3: 安装 Netlify CLI
```bash
npm i -g netlify-cli
```

### 步骤 4: 登录 Netlify
```bash
netlify login
```

### 步骤 5: 部署
```bash
netlify deploy --prod
```

### 步骤 6: 添加自定义域名
1. 访问 https://app.netlify.com
2. Site settings → Domain management
3. 添加 `woshiguotao.cn`

### 步骤 7: 配置域名解析
| 记录类型 | 主机记录 | 记录值 |
|---------|---------|--------|
| A | @ | 75.2.60.5 |
| CNAME | www | [站点名].netlify.app |

## 方案三：GitHub Pages 部署

### 步骤 1: 创建 GitHub 仓库
1. 访问 https://github.com/new
2. 创建名为 `gopen-website` 的仓库

### 步骤 2: 推送代码
```bash
cd website
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/gopen-website.git
git branch -M main
git push -u origin main
```

### 步骤 3: 启用 GitHub Pages
1. 仓库 Settings → Pages
2. Source 选择 `main` 分支
3. 保存

### 步骤 4: 添加自定义域名
1. Pages 设置中添加 `woshiguotao.cn`
2. 等待 DNS 验证

### 步骤 5: 配置域名解析
| 记录类型 | 主机记录 | 记录值 |
|---------|---------|--------|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| CNAME | www | 你的用户名.github.io |

## 域名解析操作步骤（阿里云）

1. 登录阿里云控制台
2. 进入「域名」→「解析设置」
3. 点击「添加记录」
4. 按上述表格填写信息
5. 等待 10 分钟生效

## 域名解析操作步骤（腾讯云）

1. 登录腾讯云控制台
2. 进入「域名注册」→「我的域名」
3. 点击「解析」
4. 按上述表格添加记录
5. 等待 10 分钟生效

## 国内访问优化

Vercel/Netlify 服务器在海外，国内访问可能较慢。

### 解决方案: Cloudflare CDN（免费）

1. 注册 https://dash.cloudflare.com
2. 添加站点 `woshiguotao.cn`
3. 将域名 DNS 改为 Cloudflare 提供的地址
4. 在 Cloudflare 中配置解析记录指向 Vercel/Netlify
