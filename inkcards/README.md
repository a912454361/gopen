# 万古长夜 - 国风粒子卡牌游戏平台

独立的云游戏平台，提供流畅的游戏体验。

## 功能特性

- 🎮 **云游戏**：无需下载，流畅体验
- 🎨 **国风水墨**：AI 生成水墨风格卡牌
- ⚔️ **即时对战**：匹配系统，实时对战
- 📦 **跨平台**：iOS / Android / Web / 桌面端

## 技术栈

- 前端：Expo 52 + React Native
- 后端：Express + WebSocket
- 云游戏：流式传输技术

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发
pnpm dev

# 构建 Web 版本
pnpm build:web

# 构建移动端
pnpm build:android
pnpm build:ios
```
