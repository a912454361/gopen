# G open 应用上线指南

## 一、Android APK 构建

### 方式一：EAS 云构建（推荐）

```bash
# 1. 登录 Expo 账号
eas login

# 2. 构建 APK
cd /workspace/projects/client
eas build --platform android --profile preview

# 3. 等待构建完成，下载 APK
# 构建完成后会显示下载链接
```

### 方式二：本地构建

```bash
cd /workspace/projects/client

# 生成 Android 目录
npx expo prebuild --platform android

# 构建 APK（需要 Android Studio）
cd android
./gradlew assembleRelease
```

## 三、应用商店上架所需资料

|------|------|----------|
| 软件著作权 | 国内应用商店必备 | 自行申请（约300元）或使用代办 |
| ICP 备案号 | 域名备案 | 已提交，等待审核 |
| 应用图标 | 512x512 PNG | 已有 |
| 应用截图 | 5-8张，不同尺寸 | 需要截取 |
| 应用描述 | 简介和详细描述 | 需要编写 |
| 隐私政策 | 必须 | 需要编写 |
