## 四、证书申请指南

### 软件著作权证书

#### 申请流程
1. 访问中国版权保护中心：https://www.ccopyright.com.cn
2. 注册账号并实名认证
3. 填写《计算机软件著作权登记申请表》
4. 提交以下材料：
   - 软件源程序（前30页+后30页，共60页）
   - 软件说明书（用户手册或设计说明书）
   - 身份证明文件
   - 软件鉴别材料

#### 时间周期
- 普通申请：2-3个月
- 加急申请：3-5个工作日（费用约1000元）

#### 费用参考
- 官方费用：免费
- 代理服务：300-1000元
- 加急费用：500-2000元

#### 准备材料
- [ ] 源代码文档（格式化为Word文档）
- [ ] 软件说明书（截图+文字说明）
- [ ] 营业执照或身份证扫描件

## 五、开发者账号注册

### Google Play Console

#### 注册流程
1. 访问：https://play.google.com/console
2. 使用 Google 账号登录
3. 支付 $25 一次性注册费
4. 完成开发者资料填写
5. 等待审核（通常1-2天）

#### 所需信息
- 开发者名称
- 联系邮箱
- 联系地址
- 网站 URL（可选）

---

### Apple Developer Program

#### 注册流程
1. 访问：https://developer.apple.com/programs
2. 使用 Apple ID 登录
3. 选择"个人"或"组织"账号
4. 支付 $99 年费
5. 完成身份验证
6. 等待审核（通常1-2天）

#### 所需信息
- 真实姓名
- 联系电话
- 联系地址
- 身份证件
