## 二、申请流程（详细步骤）

### 第一步：注册账号

1. 访问中国版权保护中心官网
   ```
   https://www.ccopyright.com.cn
   ```

2. 点击右上角"注册"

3. 选择"个人用户注册"

4. 填写注册信息：
   - 用户名：自定义
   - 密码：8-16位，包含字母和数字
   - 手机号：用于接收验证码
   - 邮箱：用于接收通知
   - 真实姓名：与身份证一致
   - 身份证号：18位身份证号

5. 勾选"我已阅读并同意服务条款"

6. 点击"立即注册"

7. 完成手机验证和邮箱验证

---

### 第二步：实名认证

1. 登录后，系统会提示"实名认证"

2. 上传身份证正反面照片：
   - 正面：人像面
   - 反面：国徽面
   - 要求：清晰、完整、四角对齐

3. 填写实名信息：
   - 真实姓名
   - 身份证号
   - 有效期

4. 提交审核（通常1-2个工作日）

5. 审核通过后，账号状态变为"已实名"

---

### 第三步：填写申请表

1. 登录后，点击"我要登记"

2. 选择"计算机软件著作权登记申请"

3. 填写软件基本信息：

   **软件名称**：
   ```
   G open 智能创作助手系统 V1.0
   ```
   
   **软件简称**：
   ```
   G open
   ```
   
   **版本号**：
   ```
   V1.0
   ```
   
   **软件分类**：
   - 选择：应用软件 > 工具软件
   
   **开发方式**：
   - 选择：独立开发
   
   **开发完成日期**：
   ```
   2025年01月
   ```
   
   **首次发表日期**：
   ```
   2025年01月
   ```
   
   **发表方式**：
   - 选择：网站发表
   
   **权利取得方式**：
   - 选择：原始取得
   
   **权利范围**：
   - 选择：全部权利
   
   **软件用途和技术特点**：
   ```
   本软件是一款基于人工智能技术的智能创作助手应用，集成多个顶级AI模型，
   为用户提供对话创作、图像生成、项目管理等功能。采用前后端分离架构，
   支持Android、iOS和Web三端运行。
   ```

4. 点击"保存并下一步"

---

### 第四步：上传申请材料

#### 1. 源程序代码
- 文件：`release-package/01-软件著作权申请/source-code.doc`
- 要求：前30页 + 后30页，共60页
- 格式：Word文档（.doc/.docx）
- 注意：
  - 每页不少于50行
  - 源代码要有注释
  - 去除空行

#### 2. 软件说明书
- 文件：`release-package/01-软件著作权申请/user-manual.md`
- 要求：至少20页
- 格式：Word文档（.doc/.docx）
- 注意：
  - 包含软件功能说明
  - 包含界面截图
  - 包含操作流程

**转换步骤**：
1. 打开 `user-manual.md` 文件
2. 复制所有内容
3. 粘贴到 Word 文档
4. 调整格式：
   - 标题：黑体，三号
   - 正文：宋体，小四
   - 行距：1.5倍
   - 页边距：上下左右各2.5cm
5. 另存为 `.doc` 格式

#### 3. 身份证明文件
- 上传身份证正反面照片
- 格式：JPG/PNG
- 大小：每张不超过500KB

---

### 第五步：确认申请信息

1. 核对所有填写的信息

2. 确认上传的材料完整

3. 检查联系方式

4. 勾选"本人承诺以上信息真实有效"

---

### 第六步：提交申请

1. 点击"提交申请"

2. 系统生成申请号

3. 记录申请号（重要！）

4. 打印申请表（可选）

---

## 三、申请进度查询

### 查询方式
1. 登录官网
2. 点击"我的申请"
3. 查看申请状态

### 申请状态说明
- **待审核**：已提交，等待审核
- **补正**：材料需要修改
- **受理**：材料审核通过
- **登记成功**：已发证

## 四、时间与费用

### 普通申请
- 时间：2-3个月
- 费用：免费（官方）

### 加急申请
| 加急时间 | 费用 |
|---------|------|
| 3-5个工作日 | 约1000元 |
| 10个工作日 | 约800元 |
| 20个工作日 | 约500元 |
| 30个工作日 | 约300元 |

### 推荐方案
**建议选择加急服务（3-5个工作日，约1000元）**

**原因**：
- 节省时间，快速上线应用
- 所有应用市场都要求软著证书
- 早点拿到证书，早点提交审核

## 五、常见问题

### Q1：源代码文档不足60页怎么办？
**A**：可以补充更多源文件，或调整格式（减少每页行数）

### Q2：软件说明书不足20页怎么办？
**A**：可以添加更多功能说明、界面截图、操作流程

### Q3：审核不通过怎么办？
**A**：根据补正通知修改材料，重新提交

### Q4：证书有效期多久？
**A**：软件著作权有效期50年

### Q5：可以代理申请吗？
**A**：可以，代理费用约300-1000元
