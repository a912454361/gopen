## 📞 常见问题

### Q1: 忘记密码怎么办？

**解决方案**：
- 管理员密码固定，无法自助重置
- 如需修改密码，请联系系统管理员
- 或直接查看本文档中的密码信息

### Q2: Face ID无法使用？

**可能原因**：
1. 设备不支持Face ID
2. 未在系统设置中启用Face ID
3. 未授权APP使用Face ID

**解决方案**：
1. 检查设备是否支持Face ID
2. 前往系统设置 → Face ID与密码 → 启用
3. 前往系统设置 → 隐私 → Face ID → 允许G open

### Q3: 指纹登录失败？

**可能原因**：
1. 指纹传感器问题
2. 指纹录入质量差
3. 手指潮湿或有污渍

**解决方案**：
1. 清洁指纹传感器
2. 擦干手指
3. 重新录入指纹
4. 尝试其他手指

### Q4: 登录页面无法打开？

**可能原因**：
1. 网络问题
2. 服务未启动
3. 域名解析问题

**解决方案**：
1. 检查网络连接
2. 确认服务器正在运行
3. 检查域名是否正确解析

### Q5: 移动端无法访问？

**说明**：
- 管理后台仅支持PC端访问
- 屏幕宽度需 ≥ 1024px
- 移动端访问会显示提示信息

**解决方案**：
- 使用电脑浏览器访问
- 或将浏览器窗口调整到足够宽度
