# G Open AI助手 - Windows x64 版本

## 版本信息
- **版本**: 1.0.0
- **平台**: Windows x64
- **架构**: Electron 28.3.3

## 安装说明

### 方式一：解压即用（推荐）
1. 解压 `win-unpacked` 文件夹到任意目录
2. 双击运行 `G Open AI助手.exe`
3. 无需安装，即可使用

### 方式二：压缩包
1. 解压 `G-Open-AI-Assistant-1.0.0-Windows-x64.tar.gz`
2. 进入 `win-unpacked` 目录
3. 运行 `G Open AI助手.exe`

## 系统要求
- Windows 10 或更高版本
- 64位操作系统 (x64)
- 至少 500MB 可用磁盘空间
- 至少 4GB 内存（推荐 8GB+）

## 常见问题

### Q: 双击 exe 没有反应？
A: 请确保解压后保持文件夹结构完整，不要单独运行 exe 文件。

### Q: 提示缺少 DLL 文件？
A: 请安装最新版本的 Visual C++ Redistributable：
https://aka.ms/vs/17/release/vc_redist.x64.exe

### Q: 应用无法联网？
A: 请检查防火墙设置，允许 `G Open AI助手.exe` 访问网络。
