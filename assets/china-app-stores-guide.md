## 🏪 各平台提交指南

### 一、华为应用市场（优先推荐）

#### 平台信息
- **网址**：https://developer.huawei.com/consumer/cn
- **注册费用**：免费
- **审核时间**：1-3 天
- **市场份额**：国内最大 Android 分发渠道

#### 提交步骤

**第一步：注册开发者账号**
1. 访问华为开发者联盟官网
2. 点击"注册"，选择"个人开发者"
3. 填写真实姓名、身份证号
4. 上传身份证正反面照片
5. 绑定手机号和邮箱
6. 等待实名认证（通常 1-2 天）

**第二步：创建应用**
1. 登录后进入"应用服务" → "AppGallery Connect"
2. 点击"我的应用" → "创建应用"
3. 填写应用信息：
   - 应用名称：G open - AI创作助手
   - 应用包名：com.gopen.app
   - 应用类型：工具 > 效率
   - 应用图标：上传 512x512 PNG
4. 点击"确定"创建应用

**第三步：上传应用文件**
1. 进入应用详情页
2. 点击"版本信息" → "上传软件包"
3. 上传 APK 文件（从 EAS Build 下载）
4. 等待系统自动解析应用信息

**第四步：填写应用信息**

**基本信息**：
- 应用名称：G open - AI创作助手
- 应用简介：AI驱动的智能创作助手，支持对话、图像、项目管理
- 应用描述：（详见上方详细描述模板）
- 应用分类：工具 > 效率
- 内容分级：根据问卷填写

**图标和截图**：
- 应用图标：512x512 PNG
- 应用截图：至少 4 张，最多 8 张
  - 建议包含：首页、模型市场、会员中心、项目管理
- 宣传图：1024x500 PNG（可选）

**隐私政策**：
- 隐私政策链接：https://woshiguotao.cn/privacy-policy
- 权限说明：详细说明各权限用途

**资质信息**：
- 软件著作权证书：上传扫描件
- ICP 备案号：填写备案号（等待备案通过）

**第五步：提交审核**
1. 确认所有信息填写完整
2. 点击"提交审核"
3. 等待审核结果（1-3 天）
4. 如有反馈，及时修改并重新提交

#### 审核要点
- ✅ 功能与描述一致
- ✅ 无严重 Bug 和崩溃
- ✅ 隐私政策完整
- ✅ 权限使用合理
- ✅ 内容健康合规

## ✅ 审核通过检查清单

### 必查项目
- [ ] 应用名称与软著证书一致
- [ ] 应用包名正确
- [ ] 应用图标清晰
- [ ] 应用截图真实
- [ ] 应用描述准确
- [ ] 隐私政策可访问
- [ ] 软著证书上传
- [ ] ICP 备案号填写

### 常见审核问题

**问题1：隐私政策不完整**
- 解决：确保隐私政策包含数据收集、使用、共享等完整内容

**问题2：权限使用不合理**
- 解决：在隐私政策中详细说明每个权限的用途

**问题3：应用崩溃或 Bug**
- 解决：测试不同设备，确保稳定性

**问题4：内容违规**
- 解决：确保应用内无违法违规内容

**问题5：资质不全**
- 解决：补充软著证书、ICP 备案号等

## 💡 提交技巧

### 提高审核通过率的建议

1. **完整填写信息**
   - 所有必填项必须填写
   - 选填项尽量填写完整

2. **真实反映功能**
   - 截图展示真实功能
   - 描述与应用功能一致

3. **提前准备资质**
   - 软著证书提前申请
   - ICP 备案提前完成

4. **及时响应反馈**
   - 每天查看审核状态
   - 收到反馈后立即修改

5. **多平台同步提交**
   - 不要等待一个平台通过再提交下一个
   - 同时提交多个平台，节省时间
