## 第一章 软件概述

### 1.1 软件简介

G open 智能创作助手是一款基于人工智能技术的移动应用软件，专注于游戏和动漫内容创作领域。软件集成了大语言模型、图像生成、语音合成等AI能力，为创作者提供全方位的智能创作辅助。

本软件采用前后端分离架构，前端使用 Expo + React Native 技术栈开发，后端使用 Express.js 框架，数据库采用 PostgreSQL，支持跨平台运行于 Android 和 iOS 设备。

### 1.2 主要功能

- **AI对话创作**：智能对话生成游戏剧情、角色设定、关卡设计
- **图像生成**：AI生成游戏素材、角色立绘、场景插画
- **语音合成**：文字转语音，支持多种音色，可调整语速语调
- **模型市场**：分享和下载AI模型，支持游戏、动漫、语音等分类
- **云存储**：支持多种云盘存储方案，百度网盘、阿里云盘等
- **会员系统**：三级会员体系，满足不同用户需求
- **推广系统**：推广赚钱，邀请好友获得佣金奖励

### 1.3 软件特点

1. **暗黑科技风格界面**：采用深邃的暗色调设计，科技感十足
2. **智能AI能力集成**：整合多种AI服务，一键调用
3. **跨平台支持**：同时支持 Android 和 iOS 系统
4. **实时数据同步**：云端存储，多设备数据同步
5. **安全可靠**：数据加密传输，用户隐私保护

## 第三章 功能说明

### 3.1 AI对话功能

AI对话功能是本软件的核心功能之一，用户可以通过自然语言与AI进行交互，获取创作灵感和内容生成。

**功能特点：**
- 支持多轮对话，上下文关联
- 支持游戏剧情生成
- 支持角色设定创建
- 支持关卡设计建议

**界面说明：**

主界面分为三个区域：
- 顶部：显示对话标题和历史记录入口
- 中部：对话内容展示区
- 底部：输入框和发送按钮

**操作步骤：**
1. 打开应用，进入"创作"页面
2. 在输入框输入创作需求
3. 点击发送按钮
4. AI将智能生成相关内容
5. 用户可以继续对话优化结果

### 3.2 图像生成功能

图像生成功能支持通过文字描述生成图片，可用于游戏素材、角色立绘等创作场景。

**功能特点：**
- 支持多种风格：写实、动漫、像素等
- 支持多种尺寸：512x512、1024x1024、2048x2048
- 支持批量生成
- 支持图片下载保存

**操作步骤：**
1. 进入"创作"页面，选择"图像生成"
2. 输入图片描述（如：游戏角色立绘，战士职业，全身像）
3. 选择风格和尺寸
4. 点击生成按钮
5. 等待生成完成后可下载或分享

### 3.3 语音合成功能

语音合成功能可将文字转换为语音，支持多种音色选择。

**功能特点：**
- 支持20+种音色
- 支持语速调节（0.5x - 2.0x）
- 支持语调调节
- 支持多语言（中文、英文等）

**操作步骤：**
1. 进入"创作"页面，选择"语音合成"
2. 输入需要转换的文字内容
3. 选择音色、语速、语调
4. 点击合成按钮
5. 试听并下载音频文件

### 3.4 模型市场

模型市场提供丰富的AI模型资源，用户可以浏览、下载和使用各种模型。

**分类说明：**
- **游戏类**：游戏角色生成、剧情生成、关卡设计
- **动漫类**：动漫风格转换、角色生成、场景生成
- **语音类**：语音合成、语音识别、变声
- **图像类**：图像生成、图像编辑、风格转换

**操作步骤：**
1. 进入"模型市场"页面
2. 浏览或搜索感兴趣的模型
3. 点击模型查看详情
4. 下载并使用模型

### 3.5 会员系统

软件提供三级会员体系，满足不同用户的使用需求。

**会员等级说明：**

| 等级 | 价格 | 权益 |
|------|------|------|
| 免费用户 | 免费 | 每日10次对话，基础模型访问 |
| 普通会员 | ¥29/月 | 无限对话，高级模型，图像生成 |
| 超级会员 | ¥99/月 | 全部权益，语音合成，GPU租赁 |

**购买流程：**
1. 进入"会员中心"页面
2. 选择会员套餐
3. 扫码支付
4. 填写转账流水号
5. 等待审核通过

### 3.6 推广系统

推广系统允许用户成为推广员，通过邀请好友注册消费获得佣金奖励。

**推广规则：**
- 推广员可获得被推广用户消费金额的10%作为佣金
- 佣金可在达到最低提现金额后申请提现
- 提现审核通过后转入推广员账户

**操作步骤：**
1. 进入"设置" -> "推广中心"
2. 申请成为推广员
3. 复制专属推广链接
4. 分享给好友
5. 好友通过链接注册并消费后获得佣金
6. 达到提现门槛后申请提现

## 第四章 操作指南

### 4.1 软件安装

**Android系统安装：**
1. 从应用商店搜索"G open"
2. 点击"安装"按钮
3. 等待下载完成
4. 点击"打开"启动应用

**iOS系统安装：**
1. 打开App Store
2. 搜索"G open"
3. 点击"获取"按钮
4. 验证身份后自动安装
5. 安装完成后点击"打开"

### 4.2 账号注册与登录

**新用户注册：**
1. 打开应用，点击"注册账号"
2. 输入手机号码
3. 点击"获取验证码"
4. 输入收到的6位验证码
5. 设置昵称（可选）
6. 点击"完成注册"

**用户登录：**
1. 打开应用，点击"登录"
2. 输入手机号码
3. 点击"获取验证码"
4. 输入验证码
5. 点击"登录"进入主界面

### 4.3 基本操作流程

**AI对话操作：**
1. 启动应用，自动进入首页
2. 底部导航栏点击"创作"
3. 在输入框输入问题或需求
4. 点击发送按钮获取AI回复
5. 可继续对话或开始新对话

**图像生成操作：**
1. 在创作页面切换到"图像"标签
2. 输入图片描述
3. 选择风格和尺寸
4. 点击"生成"按钮
5. 等待生成完成
6. 长按图片可保存到相册

**会员购买操作：**
1. 点击底部"我的"
2. 点击"会员中心"
3. 选择要购买的套餐
4. 扫描收款码支付
5. 支付成功后返回应用
6. 输入转账流水号
7. 提交等待审核

### 4.4 设置功能

**主题设置：**
1. 进入"设置"页面
2. 点击"主题设置"
3. 选择"亮色"、"暗色"或"跟随系统"

**通知设置：**
1. 进入"设置"页面
2. 点击"通知设置"
3. 开启或关闭各类通知

**隐私设置：**
1. 进入"设置"页面
2. 点击"隐私设置"
3. 管理数据收集偏好

## 第五章 技术特点

### 5.1 架构设计

本软件采用现代化的前后端分离架构：

**前端架构：**
- 框架：Expo + React Native
- 路由：Expo Router
- 状态管理：React Context + Hooks
- UI组件：自定义组件库

**后端架构：**
- 框架：Express.js
- 数据库：PostgreSQL (Supabase)
- 存储：对象存储服务
- AI服务：Coze API

### 5.2 安全机制

- **数据加密**：敏感数据使用AES-256加密存储
- **传输安全**：全站HTTPS加密传输
- **身份认证**：JWT Token认证机制
- **权限控制**：基于角色的访问控制

### 5.3 性能优化

- **懒加载**：图片和组件按需加载
- **缓存策略**：智能缓存减少网络请求
- **代码分割**：按路由分割减小包体积
- **内存管理**：及时释放不用的资源

## 第六章 常见问题

### 6.1 安装问题

**Q: 安装时提示"解析包错误"？**
A: 请检查系统版本是否满足最低要求（Android 8.0+），并确保存储空间充足。

**Q: iOS安装后闪退？**
A: 请尝试重启设备，如问题持续请联系客服。

### 6.2 登录问题

**Q: 收不到验证码？**
A: 请检查手机号是否正确，确认是否被拦截软件屏蔽，等待60秒后重试。

**Q: 登录后显示"账号异常"？**
A: 可能账号存在安全风险，请联系客服处理。

### 6.3 功能问题

**Q: AI生成内容很慢？**
A: 这取决于网络状况和服务器负载，请耐心等待或切换网络重试。

**Q: 图像生成失败？**
A: 请检查描述内容是否合规，避免使用敏感词汇。

### 6.4 支付问题

**Q: 支付后多久生效？**
A: 管理员审核后立即生效，一般30分钟内完成。超过30分钟自动通过。

**Q: 如何退款？**
A: 请联系客服提交退款申请。
