## 打印步骤

### 第一步：复制文件内容

1. 打开文件 `源代码文档-打印版.md`
2. 全选复制内容（Ctrl+A，Ctrl+C）
3. 打开 Microsoft Word
4. 粘贴内容（Ctrl+V）

### 第二步：调整格式

**源代码文档格式设置：**

```
字体：Consolas 或 宋体
字号：小四号（12pt）
行距：单倍行距
页边距：普通（上下左右各2.54cm）
```

**软件说明书格式设置：**

```
标题字体：黑体，三号
正文字体：宋体，小四号
行距：1.5倍
页边距：普通（上下左右各2.54cm）
```

### 第三步：添加页眉页脚

**页眉格式：**
```
第X页                          G open V1.0
```

**设置方法：**
1. Word菜单：插入 → 页眉
2. 输入：第[页码]页（按Tab键）G open V1.0
3. 插入页码：插入 → 页码 → 页面底端

### 第四步：打印

1. 文件 → 打印
2. 设置：
   - 纸张大小：A4
   - 打印方向：纵向
   - 页面范围：全部
   - 打印方式：单面打印
   - 颜色：黑白即可

3. 点击打印
