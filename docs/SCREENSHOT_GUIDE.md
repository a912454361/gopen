# 小程序截图准备指南

## 📸 截图规格要求

| 项目 | 要求 |
|------|------|
| 尺寸 | 750 x 1334 (推荐) 或更大 |
| 格式 | PNG 或 JPG |
| 大小 | 单张不超过 2MB |
| 数量 | 5 张 |
| 方向 | 竖屏截图（横屏游戏需调整展示方式） |

## 🎯 截图内容规划

### 截图1：游戏主界面

**文件名**: `screenshot_01_home.png`

## 🛠️ 截图制作方法

### 方法一：微信开发者工具截图

1. 打开微信开发者工具
2. 选择要截图的页面
3. 点击「工具」→「截图」
4. 选择保存位置

### 方法二：真机截图

1. 手机扫码预览小程序
2. 横屏显示游戏
3. 手机截图
4. 传到电脑处理

### 方法三：模拟器截图

1. 使用微信开发者工具模拟器
2. 调整设备尺寸
3. 使用系统截图工具

## ✂️ 截图后期处理

### 必要处理
1. **尺寸调整**: 确保符合750x1334或更大
2. **格式转换**: 导出为PNG或JPG
3. **压缩优化**: 单张不超过2MB

### 可选处理
1. **添加标注**: 突出核心功能
2. **添加边框**: 统一视觉风格
3. **添加文字**: 简短功能说明

### 推荐工具
- **Photoshop**: 专业图片处理
- **Sketch**: Mac用户首选
- **Figma**: 在线协作工具
- **TinyPNG**: 图片压缩

## 🎨 截图风格建议

### 色调
- 以游戏主题色为主（紫色、金色）
- 保持整体风格统一
- 避免过于花哨

### 构图
- 主体居中或黄金分割
- 信息分布均衡
- 留白适当

### 文字
- 字体清晰
- 大小适中
- 颜色对比明显

## ⚠️ 注意事项

1. **禁止内容**
   - 不得包含其他平台Logo
   - 不得有敏感文字
   - 不得侵犯他人版权

2. **真实展示**
   - 截图必须来自实际运行效果
   - 不得过度美化虚假宣传
   - 功能需与实际一致

3. **隐私保护**
   - 隐藏真实用户信息
   - 使用测试数据
   - 避免显示UID等敏感信息
